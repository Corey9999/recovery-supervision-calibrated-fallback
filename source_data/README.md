# Generated source data

This directory contains generated and derived artifacts used to audit the reported experiments and regenerate analyses, tables, and figures. It does not contain the redistributed raw UCI or Figshare datasets.

The files include:

- deterministic split and controlled-fault manifests;
- seed-, condition-, class-, and fault-level metrics;
- prediction-level outputs for paired analyses;
- bootstrap and paired-test results;
- calibration, selective-risk, teacher-agreement, and QMF sensitivity audits;
- compact NumPy arrays generated by the synthetic benchmark.

Large prediction CSV files are retained for auditability. The largest individual file remains below GitHub's 100 MiB hard limit, but command-line Git or GitHub Desktop is required because the browser uploader has a lower per-file limit.

The original public datasets can be downloaded with `python download_public_data.py`; see `../DATASETS.md` for source DOIs and licence information.
